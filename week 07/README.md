## Двумерни масиви

### Дефиниция на двумерен масив
**Свойства:**
- масив от масиви
- индексация на подмасивите, започвайки от `0` до `rowSize - 1`
  - индексация на елементите на всеки подмасив, започвайки от `0` до `colSize - 1`

**Дефиниция:**
<тип> <име>[<брой_редове>][<брой_колони>];

*Пример:* 
```c++
int a[5][7];
```

<име>[<индекс_ред>][<индекс_колона>] = <стойност>;

*Примери:* 
```c++
a[0][0] = 2;
int x = a[3][1];
```
