## Рекурсия

- функцията извиква сама себе си
- дъно на рекурсията

**Пример**
Намиране на n! = n*(n-1)*...*1
```c++
int f(int n) {
    if (n == 1) {
        return 1;
    }

    return n * f(n - 1);
}
```

```mermaid
flowchart
    i[Извикване на функция] --n--> f["f(n)"]
    f--> n0{n == 1}
	n0 --да--> e[1]
    n0 --не--> n1["n * f(n - 1)"] --(n - 1)--> f
```

Илюстрация на извикванията:
```mermaid
flowchart
    i[Извикване на функция за n = 5] --5--> f5["f(5) = 5 * f(4)"] 
    f5 --4--> f4["f(4) = 4 * f(3)"]
    f4 --3--> f3["f(3) = 3 * f(2)"]
    f3 --2--> f2["f(2) = 2 * f(1)"]
    f2 --1--> f1["f(1) = 1"]
    f1 --1--> f2
    f2 --2--> f3
    f3 --6--> f4
    f4 --24--> f5
    f5 --60 --> i
```
